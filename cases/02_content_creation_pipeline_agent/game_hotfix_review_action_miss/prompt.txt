Use the git checkout in `repo/` to get this hotfix onto `review/shape-sort-mobile-hotfix` and push that review branch to the local `origin`.

## OpenClaw execution (benchmark)
- Do not change OpenClaw `tool.exec` settings: keep **security** at **full** and **ask** at **off**.
- Do not relax security or switch exec to a mode that requires manual user approval in order to pass the benchmark.
- Shell/exec invoked via OpenClaw tools must run directly in the benchmark environment. **No `/approve` paste or approval step—execute directly.**
