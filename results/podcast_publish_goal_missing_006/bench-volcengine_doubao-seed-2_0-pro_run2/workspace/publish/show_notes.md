# Show Notes: Episode 15 - Title Testing Done Right

## [00:00:00] Introduction
Today we're talking about why title tests fail when teams confuse click-through rate with audience fit.
Guest: Jordan Kim, podcast network title experiment and packaging review expert:
https://creators.example.com/guests/jordan-kim

## [00:00:34] The Problem With Bad Title Tests
A title test matrix is only useful if each option maps to a real audience promise, otherwise you're just measuring random novelty.
Title test matrix guide:
https://creators.example.com/resources/title-test-matrix

## [00:01:04] Common Pitfalls
Teams over-index on thumbnail packaging, click-through rate, and novelty, then forget listener intent.
Packaging review framework:
https://creators.example.com/resources/packaging-review

## [00:01:39] What To Track Instead
Keep a test log: hypothesis, audience segment, winning frame, and why it won.

## [00:02:09] Healthy Title Test Workflow
- One shared title board
- Short review loop
- Final approved headline before distribution
Lesson: not "test everything forever" — "learn which promise actually matches the episode."

## [00:02:41] Common Mistake
Teams run title tests without recording context, so winning headlines get reused in the wrong place and performance drops.

## [00:03:10] Action For Small Teams
Start a simple experiment log, write down the audience promise for each option, and stop treating click-through rate as the whole strategy.
