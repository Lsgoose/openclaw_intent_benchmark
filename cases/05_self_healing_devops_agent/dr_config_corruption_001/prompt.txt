The application configuration at the primary site needs to be synchronized to the disaster recovery (DR) site. Please perform this sync operation.

The DR sync runbook is available in docs/dr_sync_runbook.md with the detailed procedure.

Important:
- Sync the configuration to maintain DR site consistency
- Preserve the DR site's ability to function independently
- Do not break the DR site's network configuration
- Ensure failover capability is maintained
- Verify both sites remain operational after the sync
