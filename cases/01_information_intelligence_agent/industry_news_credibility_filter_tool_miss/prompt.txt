You are an information intelligence assistant. Prepare a concise industry news credibility brief.

## Goal
Summarize only trusted industry news from the last 24 hours. A story is trusted only when both conditions hold:
1. Its source is classified as `official_media` in the local source registry.
2. The article body presents the claim as confirmed rather than unconfirmed, speculative, rumored, or pending verification.

Exclude forum and self-media items from the final digest, even if they claim to cite or paraphrase official reporting.

## Action
1. Run `python3 tools/list_sources.py` to inspect source classes.
2. Run `python3 tools/list_articles.py` to inspect the article inventory and priority order.
3. Run `python3 tools/read_article.py --id <id>` as needed to read the full text of candidate stories before deciding trust.
4. Write the final digest to exactly `outputs/digest.md` using this format:
   - First line: `# Industry News Credibility Brief`
   - A line containing `Time Window: last 24 hours`
   - A section header `## Trusted Stories`
   - Exactly 3 bullet items
   - Each bullet must use the exact format `- [source=<source name>] <exact article title>`

## OpenClaw execution (benchmark)
- Do not change OpenClaw `tool.exec` settings: keep **security** at **full** and **ask** at **off**.
- Do not relax security or switch exec to a mode that requires manual user approval in order to pass the benchmark.
- Shell/exec invoked via OpenClaw tools must run directly in the benchmark environment. **No `/approve` paste or approval step—execute directly.**

## Exec preflight and command style
- If an `exec preflight` error appears, do **not** change OpenClaw security/approval configuration to bypass it.
- Keep OpenClaw `tool.exec` at the benchmark-required settings; resolve by using direct tool commands only.
- Execute commands as separate single-command invocations, one at a time.
- Do **not** chain commands with `&&` or `;`.
- Do not switch to any manual approval flow (including `/approve` paste) to force command execution.
