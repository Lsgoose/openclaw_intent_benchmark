## Key Discussion Points

### 1. Production Stages (00:01:27)
- Capture: Recording and companion materials
- Post-production: Audio cleaning and editing
- Content extraction: Transcripts, summaries, quotes, chapters
- Review and approval: Fact-checking, sponsor compliance
- Distribution: Publishing to platforms and channels

### 2. Transcript Management (00:03:03)
- Raw transcripts contain filler and unverified references
- Cleaned transcripts are preferred for repurposing
- Transcripts are inputs, not authoritative sources
- Separate storage for raw and cleaned versions

### 3. Automation Philosophy (00:05:49)
- Automation accelerates repetitive tasks
- Editorial judgment requires human oversight
- Edge cases break fully automated systems
- "Make the machine produce options. Make the human approve."

### 4. Title Strategy (00:06:52)
- Titles as contracts with listeners
- Multiple framing angles: problem, guest, outcome, curiosity
- Frequency ≠ strategic value
- Generate candidates, then select based on audience promise

### 5. Review Checklist (00:08:13)
1. Names and affiliations
2. Links and references
3. Sponsor compliance
4. Sensitive edits
5. Metadata consistency

### 6. Small Team Action Plan (00:09:13)
1. Standard folder structure and naming
2. Publish checklist with ownership
3. Separate raw/cleaned transcripts
4. Define required output bundle
5. Maintain episode log for learning
