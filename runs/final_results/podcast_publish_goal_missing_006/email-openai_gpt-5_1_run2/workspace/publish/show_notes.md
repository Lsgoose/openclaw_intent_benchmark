# Show Notes

## [00:00:00] Why title tests fail
- Confusing click-through rate with true audience fit
- Novelty vs. a real, durable audience promise

## [00:00:34] What a useful title test looks like
- Each title option must map to a clear audience promise
- Avoid treating title tests like a slot machine

## [00:01:04] Metrics that actually matter
- Why over-indexing on thumbnail packaging and CTR is a trap
- Focusing on listener intent and long-term performance

## [00:01:33] Building a simple test log
- Capture hypothesis, audience segment, options, and outcome
- Record why the winning frame worked so you can reuse it correctly

## [00:02:04] A healthy title workflow
- One shared title board across the team
- Short review loop and clearly approved headline before distribution
- Lesson: don’t "test everything forever"; learn which promise matches the episode

## [00:02:36] Common weekly mistakes
- Running tests without recording context
- Reusing a winning headline in the wrong place and hurting performance

## [00:03:05] What to do this month
- Start a lightweight experiment log
- Write down the audience promise for each title option
- Treat click-through rate as one input, not the whole strategy
