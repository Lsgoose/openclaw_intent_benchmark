# Episode 14: Podcast Production Pipeline — with Maya Chen

Media advisor Maya Chen joins the show to unpack how small teams can build reliable, consistent production systems. The conversation maps the five stages of a healthy pipeline—capture, post-production, content extraction, review, and distribution—and identifies where teams typically break down: messy handoffs, inconsistent file naming, and treating raw transcripts as publish-ready truth.

Chen makes the case that good pipelines reduce noise before it reaches the next tool or teammate. She distinguishes between automation that accelerates (first drafts, formatting, metadata prefill) and automation that abdicates (editorial judgment, exception handling). The episode also covers practical tactics: maintaining both raw and clean transcripts, standardizing output bundles per episode, and treating titles as contracts with listeners rather than mere labels.

The closing segment delivers a five-point action plan any small show can implement this month without elaborate tooling—proving that consistency, not complexity, drives production quality.
