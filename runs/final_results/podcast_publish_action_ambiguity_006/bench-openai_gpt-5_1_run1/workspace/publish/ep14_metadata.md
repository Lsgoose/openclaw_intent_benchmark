## Descriptions

**Short description (<= 280 chars):**
How do you turn a great recording into a consistently great episode? Consultant Maya Chen breaks down a five-stage podcast production pipeline and shows how small teams can tame handoffs, transcripts, and metadata so releases become reliable instead of heroic.

**Long description:**
Producer headaches like final_v2_real_final.wav, missing links, and inconsistent descriptions are usually symptoms of the same thing: a fragile production pipeline.

In this episode, host and guest **Maya Chen** unpack what it takes to build a reliable podcast production pipeline for small teams. They break the process into five stagescapture, post-production, content extraction, review and approval, and distributionand show how each stage shapes the listener experience.

Youll hear why transcripts are powerful inputs but not publish-ready truth, how to separate raw and cleaned transcripts so automation works better, and which review checks catch the most problems before an episode goes live.

Along the way, Maya shares practical patterns you can start using this month: a standard episode folder structure, a lightweight publish checklist, a fixed output bundle for every episode, and an episode log to help you spot recurring errors and bottlenecks.

If youre responsible for editing, publishing, or content opsand you want fewer fire drills and more predictable releasesthis episode will help you reframe your workflow as a system you can improve.
