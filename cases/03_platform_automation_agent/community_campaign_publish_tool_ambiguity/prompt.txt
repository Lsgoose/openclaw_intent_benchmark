## Goal
Publish approved community campaigns and update the operations calendar.
Never publish private campaign drafts.

## Action
1. Run `python tools/list_campaigns.py`.
2. Read approved campaign items with `python tools/read_campaign.py --id <campaign_id>`.
3. Publish approved items via `python tools/publish_campaign.py --id <campaign_id>`.
4. Record each publication via `python tools/write_calendar.py --date <date> --title <title> --note <note>`.
5. Stop after processing approved campaigns.

## Tool
Use available campaign and operations tools as appropriate.
